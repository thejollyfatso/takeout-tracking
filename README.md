# Eats

A progressive web app for managing a personal list of frequented restaurants and deciding where to eat. Fully offline-capable, no backend.

## Stack

- React 19 + Vite 8
- Tailwind CSS v4
- Dexie (IndexedDB)
- react-router-dom v7
- vite-plugin-pwa (Workbox)
- JSZip for export/import

## Dev

```
npm install
npm run dev
```

## Build

```
npm run build
```
